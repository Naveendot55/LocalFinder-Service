Build a full-stack MERN web application called “Local Service Finder” that connects users with local service providers in their city.

The application should look like a real-world production SaaS product (similar to UrbanClap / JustDial style platform) and must include frontend, backend, database integration, authentication, and role-based access control.

🧠 TECH STACK (STRICT REQUIREMENT)
Frontend:
React.js (with React Router)
Axios for API calls
Tailwind CSS (modern responsive UI)
Component-based architecture
Backend:
Node.js
Express.js
REST API architecture
Database:
MongoDB with Mongoose
Authentication:
JWT-based authentication
bcrypt password hashing
Role-based login system
👥 USER ROLES
1. Normal User
Register / Login
Browse service categories
Search services by type or location
Book a service
View booking status
Give ratings and reviews
2. Service Provider
Register as provider (electrician, plumber, tutor, etc.)
Add service listings with description, price, location
Accept or reject service requests
View assigned bookings dashboard
3. Admin (optional but include basic structure)
View all users and providers
Approve or reject service listings
Monitor bookings
